# 반응형 웹?

디바이스의 크기와 viewport의 크기를 고려하여 어떤 크기에서도 서비스를 화면에 보여주는 웹

<br>


## viewport?
viewport는 display에서 웹을 보여주고 있는 화상표시 영역.

width : 화면 너비를 따르도록 페이지 너비를 조정한다.

initial-scale : 브라우저에서 페이지가 첫 로드 했을때 초기 확대 || 축소 수준을 뜻한다.

minimum-scale : 최소 화면 배율을 설정

maximun-scale : 최대 화면 배율 설정


## 미디어 쿼리

viewport의 width, height값에 따라 구간별로 다른 CSS속성을 적용해주는 일종의 조건문 같은 역할.

이는 반응형을 만드는 데 있어서 중요한 역할을 한다.

<br>

## 가변그리드

화면의 크기에 상관없이 확대, 축소할 수 있도록 px 대신 % 단위로 코드를 작성하는 기술

> 공식 : (가변크기로 만들 가로 / 가변크기로 만들 세로) * 100 = 가변 크기의 % 값)

<br>

## 가변폰트

em,rem , vw,vh,등등으로 px대신 사용하여 자유롭게 크기를 지정해줌
