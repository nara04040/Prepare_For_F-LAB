# 변수선언

기본적인 변수 선언식

```javascript
var a;
```

**"변할 수 있는 데이터를 만든다. 이 데이터의 식별자는 a로 한다"** == **"변경 가능한 데이터가 담길 수 있는 공간 또는 그릇"**


<br>

var a; 으로 변수를 선언 한다면 위 그림과 같이 1003공간에 a를 지정한다.
